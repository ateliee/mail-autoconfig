mail-autoconfig
===============

メールの設定情報を自動検知する為の設定ファイル

## 使い方
.well-knownフォルダを  
http://example.com/.well-known/autoconfig/mail/config-v1.1.xml でアクセスできる箇所に、置きます。(example.comは自分のドメイン名)  
基本的にはそのままでも動きますが、設定が必要な場合はconfig.phpの中身を設定致します。
